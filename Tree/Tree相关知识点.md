# 树相关知识点

## 二叉树
- 二叉树是由有限个节点的集合组成的树，这个集合或者为空，或者是由一个根节点和两个互不相交的左子树和右子树组成的二叉树构成
- 每个树的节点可以是数字，可以是Linked List，也可以是List等等

## 满二叉树
- 二叉树的一种特殊形式，其中只有双分枝节点和叶子节点，并且所有的叶子节点在同一层

## 完成二叉树
- 最多只有下面2层的节点的度可以小于2，并且最下面一层的叶子节点都依次排列在该层最左边的位置上

## 二叉树的前序遍历
- 先访问根结点-再访问左子树-再访问右子树

## 二叉树的中序遍历
- 先访问左子树-再访问根结点-再访问右子树

# 二叉树的后序遍历
- 先访问左子树-再访问右子树-再访问根结点

# 二叉树的层次遍历
- 先访问根节点
- 从左到右访问第二层的所有节点 
- 从左到右访问第三层的所有节点 
- 以此类推
